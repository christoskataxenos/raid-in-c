# Οδηγίες για την "Proper" έκδοση (Multi-file)

Εδώ είναι ο κώδικας που είναι σπασμένος σε πολλά αρχεία. Είναι πιο σωστό έτσι για μεγάλες εργασίες.

## Τι περιέχει ο φάκελος:
- `main.c`: Η αρχή του προγράμματος.
- `shell.c / .h`: Το μενού με τις επιλογές (1-8).
- `raid.c / .h`: Εδώ γίνεται όλη η δουλειά (XOR, RAID 0, 1, 5).
- `disk.c / .h`: Διαχείριση των αρχείων για τους δίσκους.
- `env.c / .h`: Για να κάνουμε reset και backup.

## Πώς το τρέχουμε

### Windows:
Τρέξε το `compile.bat` ή γράψε:
```cmd
gcc main.c shell.c raid.c disk.c env.c -o raid.exe
```

### Linux / WSL:
```bash
gcc main.c shell.c raid.c disk.c env.c -o raid
./raid
```

## Μερικές σημειώσεις:
1. Το μενού είναι νούμερα @1-8@, δεν χρειάζεται να γράφετε εντολές.
2. Αν ένας δίσκος χαλάσει (`FAIL`), το RAID 5 θα συνεχίσει να δουλεύει κανονικά (κάνει το parity XOR στον αέρα).
3. Με το `RECOVER` φτιάχνετε τον δίσκο που χάλασε.

*Έγινε για Windows 2000.*
